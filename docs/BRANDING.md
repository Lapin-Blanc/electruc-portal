# Electruc — Mini charte graphique (v1)

## Intention
Identité institutionnelle sobre, avec un humour discret via l’icône « éclair-sourire ».
Objectif : inspirer confiance, rester lisible pour un public senior.

## Couleurs
- Accent (vert énergie) : `#6FCF97`
- Encre (titres/nav) : `#243447`
- Texte : `#1F2937`
- Fond doux : `#F4F6F8`
- Bordures : `#D9DEE6`

## Typographie
- Police : `system-ui` (par défaut), lisible et disponible partout.
- Titres : gras (700–800), mêmes familles de police.

## Utilisation du logo
- Logo principal : `electruc-logo.svg` (préféré) ou `electruc-logo.png`.
- Icône : `electruc-icon.svg` / `electruc-icon-512.png` (favicon, onglet).
- Version monochrome : `electruc-logo-mono.svg`.

## Ton rédactionnel (microcopy)
- Simple, rassurant, concret (public senior).
- Éviter le jargon : dire “Télécharger la facture (PDF)” plutôt que “Exporter”.
- Une seule action principale par page.

## Statuts (recommandation)
- Facture : À payer / Payée
- Domiciliation : En attente / Active / Refusée
- Demande : Ouverte / Répondue / Clôturée
