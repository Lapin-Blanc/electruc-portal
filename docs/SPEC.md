# Spécification - Electruc Portal

## Objectif
Créer une application web pédagogique simulant un portail client de fournisseur d'énergie (Belgique).

## Public cible
- Seniors
- Adultes en formation
- Initiation aux usages numériques

## Principes
- Vocabulaire métier (sans jargon technique inutile)
- Interface sobre et institutionnelle
- Une action principale par page
- Messages clairs, sobres et rassurants
- Données fictives mais cohérentes

## Contraintes
- Aucune référence à une entreprise réelle
- Aucune fonctionnalité de paiement réel
- Projet volontairement simple, lisible et maintenable

## Scénario pédagogique principal
- Import de points de fourniture (EAN)
- Génération d'invitations papier
- Inscription autonome client via EAN + code d'activation unique
- Activation par email
- Consultation contrat, factures, relevés, demandes
